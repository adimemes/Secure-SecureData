# SecureData And Cleaning Data Slang Word 


# Overview

I'am build this app using Python. The main functionality of this file is to Secured and Cleaner a sentence in Indonesian. used Regex to implement the Secured data and use Dataset Slangwords indonesia to create Cleaner Sentence. the input is a sentence in Indonesia, and the output is the Secured and Cleaner status of the inputted sentence.


# Requirement

If you want to install it locally you need to install a few of libraries for the app:

You can install requirements.txt by executing this command in your project directories terminal:
```

    pip install requirements.txt

```

After that, you can run the app using streamlit like this in your project directories terminal
```

  streamlit run app.py

```
